# vue.js 全家桶仿网易云音乐 webApp

## 本项目是使用vue.js 实现的 网易云音乐webapp，功能架构有参考开源项目react hook 音乐webapp，主要功能均已实现

### 主要技术栈 vue、vuex、sass 、帧动画等，仅供参考
